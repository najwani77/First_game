## Run locally
1) Install Node.js LTS: https://nodejs.org
2) Open terminal in this folder and run:
   npm install
   npm run dev
3) Open http://localhost:3000
4) Start the story at /episode/ep1

## Add episodes
- Put JSON files into /public/episodes/
- See /public/episodes/ep1.json as an example.
- Add backgrounds to /public/bg, character sprites to /public/chars.

## Build for production
npm run build && npm start
